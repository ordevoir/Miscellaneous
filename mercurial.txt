все комманды начинаются с hg

создание репозитория:

переходим в терминале в папку с проектом

hg init		- создает репозиторий

в папке с проектом появится скрытая папка .hg

hg status	- показывает состояние репозитория

hg add		- добавление файлов в репозиторий
git add .	- (можно использовать для добавления всех изменений сразу)

hg commit	- фиксирование изменений (сохраняет текущее состояние всех файлов в репозиторий)

hg log		- отображает историю изменений, зафиксированных в репозитории

изменения, сделанные до фиксирования можно просмотреть при помощи diff, и отменить при помощи revert:

hg diff		- просмотреть изменения

git show [code] - показывает изменения фиксации с кодом code

hg revert -all	- отмена всех изменений
hg revert file 	- отмена изменений в конкретном файле

hg remove file	- помечает файлы как запланированные для удаления из репозитория

hg cat file 	- просмотр содержимого файла
hg cat -r i file	- просмотр i-ой версии файла
hg diff -r i:j file	- просмотри различия между i-ой и j-ой версиями файла

hg update -r i	- переход к i-ой версии (r - revision)

hg clone

hg outgoing	-отображает список изменений ждущих отправки в текущем репозитории
